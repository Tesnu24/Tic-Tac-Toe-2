# 🎮 Tic Tac Toe – Solo or Duo

An interactive and responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.
Play with a friend or challenge a smart computer opponent.

---

## ✨ Features

* 🧑‍🤝‍🧑 Two Player Mode
* 🤖 Single Player Mode (smart AI: win, block, best move)
* ✍️ Custom player names
* 🟩 Responsive 3×3 game board
* 🏆 Score tracking (X wins, O wins, Draws)
* 🔄 Restart round & Reset game
* 🎨 Clean UI with hover effects and visual feedback

---

## 🧠 Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (DOM manipulation & game logic)

---

## 🎯 Game Logic

* Players take turns placing **X** and **O**
* A player wins by getting **3 in a row (row, column, or diagonal)**
* If all cells are filled with no winner → **Draw**
* In single-player mode, the computer:

  * Tries to win
  * Blocks the player
  * Chooses the best available move
